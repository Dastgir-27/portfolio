# Dastgir's Portfolio

A personal portfolio website built to showcase my projects, skills, and professional experience.

## ✨ Features

- **Modern & Responsive UI**: Designed with a sleek, mobile-first approach.
- **Smooth Animations**: Powered by [GSAP](https://gsap.com/) and [Framer Motion](https://www.framer.com/motion/) for fluid and engaging user experiences.
- **Component Library**: Utilizing [shadcn/ui](https://ui.shadcn.com/) and [Radix UI](https://www.radix-ui.com/) for accessible and customizable components.
- **Fast & Optimized**: Built with [Vite](https://vitejs.dev/) for lightning-fast HMR and optimized production builds.

## 🛠️ Tech Stack

- **Framework**: [React 18](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Animations**: GSAP, Framer Motion

## 🚀 Getting Started

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Dastgir-27/portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio/p
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`.

## 📦 Build for Production

To create a production-ready build, run:
```bash
npm run build
```
This will compile and minify your application into the `dist` folder.
